# Legacy Security Overlay Context

The repository retains an earlier PaymentOps hardening overlay for release
manifest authentication, bundle verification, cross-file invariants, and
verified-byte loading.

Historical v0.8.2 qualification files may remain in local working directories.
They are evidence for a standalone security-overlay scope and are not the
evidence basis for the v0.9.4 AML model-performance claims.

The current public model claims are limited to the v0.9.4 protocol, model card,
and frozen evidence summary. GitHub-hosted v0.9.4 Medium-data execution and
production-bank deployment are not claimed.
